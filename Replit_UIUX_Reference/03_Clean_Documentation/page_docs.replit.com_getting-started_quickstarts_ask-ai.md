## [​](#before-you-begin) Before you begin
You’ll need a free [Replit account](https://replit.com).
This quickstart uses Agent, which is included in the free Starter plan with daily credits. No credit card required.
## [​](#welcome-to-replit) Welcome to Replit
When you open Replit, the first thing you see is your **Project Editor**, with the prompt box at the center. This is where everything starts. Describe what you want to build, and Agent takes it from there.
![The Replit Project Editor home screen showing the prompt input where you describe what you want to build](https://mintcdn.com/replit/j4of5To8jzehz0JK/images/workspace/workspace-home.png?fit=max&auto=format&n=j4of5To8jzehz0JK&q=85&s=4c36166a8a0951aca0e861480dc6b17d)
## [​](#build-your-app) Build your app
1
Describe what you want to build
In the prompt box on the home screen, describe what you want to build. Be specific about features and behavior — the more detail you give Agent, the better the result.For this quickstart, try something like: *“Build a realistic 16-key piano with proper white and black key layout. Each key should press down with animation when clicked and play the correct note.”*Press **Start** to kick off the build. Agent sets up your project and starts writing code. This takes about 3–5 minutes.
You can build any idea you like — a to-do list, a landing page, a quiz app. The rest of the steps work the same way.
2
Preview your app
When Agent finishes, navigate to the **Preview** tab to see your app in action. Click around and interact with it to make sure it works the way you expect.
[](https://cdn.replit.com/sanity/quickstart-piano-demo.mp4)
Your app might look slightly different from the screenshots — Agent can generate varied results from the same prompt.
3
Publish your app
Ready to share? Click **Publish** in the Project Editor to deploy your app to a public URL that anyone can visit.
[](https://cdn.replit.com/sanity/quickstart-publish-app.mp4)
Your app is now live. Copy the URL and share it with anyone.
## [​](#try-it-now) Try it now
Ready to build? Press  to open Replit with the piano prompt pre-filled.
Build a realistic 16-key piano with proper white and black key layout that looks exactly like a real piano. Each key should press down with realistic depth animation when clicked and play the correct note.
