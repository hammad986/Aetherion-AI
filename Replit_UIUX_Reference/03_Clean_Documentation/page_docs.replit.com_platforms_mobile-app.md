The [Replit mobile app](https://replit.com/mobile) is a full-featured development environment optimized for phone and tablet. Build and publish apps wherever you are using all of Replit’s powerful tools, including Agent.
## [​](#what-is-the-replit-mobile-app) What is the Replit Mobile App
The Replit Mobile App brings the complete Replit experience to your iOS or Android device. You can create apps with Agent, access the Project Editor, and publish your projects—all from your mobile device. The app is designed for on-the-go development with an interface optimized for touch and mobile screens.
## [​](#getting-started) Getting started
Follow these steps to learn to install the app and create a Replit App.
### [​](#install-the-app) Install the app
To install the app on your compatible mobile device, follow these steps:
1. Navigate to the [mobile app download page](https://replit.com/mobile) and install the app for your mobile device OS.
2. Open the app and sign up for a new account or log in with an existing one.
   If you register for a new account, you must verify your email address to complete the process.
3. Create a Replit App from the home screen.
### [​](#create-your-first-app) Create your first app
Start building from the Create tab—it’s the first thing you’ll see when you open the app. Every app you create includes Agent.
Build with Agent
1. Open the **Create** tab
2. Describe what you want to build
3. Tap **Start chat**
Agent starts building your app right away.
Start from a template or GitHub
1. Open the **Create** tab
2. Tap ⋮ in the top right
3. Choose **Remix a Template** or **Import from GitHub**
4. Pick what you need
## [​](#features) Features
* **[Replit AI](/core-concepts/how-replit-works)**: Agent and AI-powered tools for building apps
* **[Project Editor](/core-concepts/project-editor)**: Full-featured development environment
* **[Publishing](/core-concepts/project-editor/app-setup/visibility)**: Share Replit Apps with zero setup
Live Activities
Follow along with Agent’s progress on the go.**Enable Live Activities:**
When Agent starts, you’ll see a notification. Tap **Allow** on your lock screen. Now you can check progress without opening Replit.**Supported Devices:**
* iPhone running iOS 18 or later
* iPad running iOS 18 or later
* Mac running macOS Tahoe or later
* Paired Apple Watch
Notifications
Get updates that matter, wherever you are.**Enable Notifications:**
Go to **Settings** and open **Notifications** (e.g. under **User**). Ensure Notifications is toggled on and tap allow if prompted.**When we’ll notify:**
* Agent needs your help
* Agent is done working
* Important billing updates
Tablet Keyboard Mode
Create on your tablet with a keyboard.**How it works:**
Connect a keyboard to your tablet for a creation experience optimized for larger screens. When you first open the Project Editor with a keyboard connected, you’ll be asked if you want to enable it.**Turn it on or off:**
Go to Account > Keyboard Settings anytime to toggle.**What you’ll see:**
A Project Editor layout designed for keyboard and larger screens.
## [​](#upgrade-and-billing) Upgrade and billing
Get Core or Pro
**Core:**
1. Go to **Account**
2. Tap **Upgrade**
3. Choose your plan
4. Tap **Continue with Core**
**Pro:** Pro isn’t available for purchase in the mobile app. It must be purchased on Replit.com.
Add more Agent usage
Core subscriptions purchased from Apple and Google include $20 in credits refreshed every month.Add a payment method for [usage-based billing](/billing/about-usage-based-billing) to use Agent beyond this limit.**On the web:**
1. Go to replit.com
2. Open **Settings** → **Account** → **Billing**
3. Add your payment info
**In the app (iOS in US only):**
1. Open **Settings** → **Billing** (or **Account** → **Billing**)
2. Add payment method
