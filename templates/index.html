    Copy this URL and paste it into your Razorpay Dashboard → Webhooks. Razorpay will call this endpoint after every payment event to keep your subscriptions in sync.

    Setup Guide

    1
    Add Razorpay API Keys
    Go to Replit Secrets and add RAZORPAY_KEY_ID and RAZORPAY_KEY_SECRET from your Razorpay Dashboard → Settings → API Keys.
    2
    Register the Webhook
    In Razorpay Dashboard → Settings → Webhooks → Add New Webhook. Paste the webhook URL shown above.
    3
    Select Events
    Enable: payment.captured and payment.failed in the webhook event checkboxes.
    4
    Save Webhook Secret
    Razorpay will generate a Webhook Secret. Copy it and add it to Replit Secrets as RAZORPAY_WEBHOOK_SECRET. Then restart the app.
    ✓
    Done — click Refresh above
    The status should update to ✅ Connected once all three secrets are present.

    Email Notifications (Optional)

    To send payment confirmation emails, add a Resend API key as EMAIL_API_KEY in Replit Secrets. Set EMAIL_FROM to your verified sender address (e.g. billing@yourdomain.com).
